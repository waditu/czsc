# tas_double_ma_V240208：双均线交叉结构信号

> 模块: `tas.rs` | 类别: `kline`

## 参数模板

`"{freq}_D{di}N{N}M{M}双均线_BS辅助V240208`

## 信号逻辑

1. 计算 `N/M` 双均线并识别交叉序列；
2. 最近三次交叉记作 `X1/X2/X3`；
3. `X3` 金叉且 `X2` 快线最高判 `多头`，死叉镜像判 `空头`。

## 信号列表示例

- `Signal('60分钟_D1N20M60双均线_BS辅助V240208_多头_任意_任意_0')`
- `Signal('60分钟_D1N20M60双均线_BS辅助V240208_空头_任意_任意_0')`

## 参数说明

- `di`：倒数第 `di` 根K线，默认 `1`；
- `N`：快线周期，默认 `20`；
- `M`：慢线周期，默认 `60`。

## 对齐说明

交叉类型与快线比较逻辑对齐 Python `tas_double_ma_V240208`。
