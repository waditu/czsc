# jcc_szx_V221111：十字线

> 模块: `jcc.rs` | 类别: `kline`

## 参数模板

`"{freq}_D{di}TH{th}_十字线`

## 信号逻辑

1. `(high-low)/|close-open| > th` 或 `close==open` 判十字线；
2. 按上下影长度细分 `蜻蜓/墓碑/长腿/十字线`；
3. 前一根强阳时追加 `北方`。

## 信号列表示例

- `Signal('60分钟_D1TH10_十字线_蜻蜓十字线_北方_任意_0')`
- `Signal('60分钟_D1TH10_十字线_墓碑十字线_任意_任意_0')`

## 参数说明

- `di`：倒数第 `di` 根K线，默认 `1`；
- `th`：十字线阈值，默认 `10`。

## 对齐说明

与 Python `jcc_szx_V221111` 一致。
