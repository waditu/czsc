# bar_big_solid_V230215：窗口最大实体中位多空信号

> 模块: `bar.rs` | 类别: `kline`

## 参数模板

`"{freq}_D{di}N{n}_MIDV230215`

## 信号逻辑

1. 在窗口内找到实体最大K线；
2. 取该K线实体中位价 `mid`；
3. 最新收盘价高于 `mid` 判 `看多`，否则 `看空`；
4. 最大实体K线按方向标注 `大阳/大阴`。

## 信号列表示例

- `Signal('日线_D1N20_MIDV230215_看多_大阳_任意_0')`
- `Signal('日线_D1N20_MIDV230215_看空_大阴_任意_0')`

## 参数说明

- `di`：倒数第 `di` 根K线，默认 `1`；
- `n`：窗口长度，默认 `20`。

## 对齐说明

最大实体与中位价定义对齐 Python `bar_big_solid_V230215`。
