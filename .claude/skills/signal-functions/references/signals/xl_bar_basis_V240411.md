# xl_bar_basis_V240411：吞没形态信号

> 模块: `xl.rs` | 类别: `kline`

## 参数模板

`"{freq}_N{n}_形态V240411`

## 信号逻辑

1. 最近两根K线构成看涨吞没时输出 `看涨吞没`；
2. 构成看跌吞没时输出 `看跌吞没`；
3. 否则输出 `其他`。

## 信号列表示例

- `Signal('60分钟_N2_形态V240411_看涨吞没_任意_任意_0')`
- `Signal('60分钟_N2_形态V240411_看跌吞没_任意_任意_0')`

## 参数说明

- `n`：最小窗口参数，默认 `2`。

## 对齐说明

与 Python `xl_bar_basis_V240411` 的吞没条件一致。
