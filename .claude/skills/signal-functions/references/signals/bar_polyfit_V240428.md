# bar_polyfit_V240428：一阶二阶拟合分类

> 模块: `bar.rs` | 类别: `kline`

## 参数模板

`"{freq}_D{di}W{w}_分类V240428`

## 信号逻辑

1. 对窗口收盘价做一阶拟合取斜率 `p1`；
2. 做二阶拟合取二次项系数 `p2`；
3. 按 `p1/p2` 符号组合输出 `加速上涨/减速上涨/加速下跌/减速下跌`。

## 信号列表示例

- `Signal('60分钟_D1W20_分类V240428_加速上涨_任意_任意_0')`
- `Signal('60分钟_D1W20_分类V240428_减速下跌_任意_任意_0')`

## 参数说明

- `di`：倒数第 `di` 根K线，默认 `1`；
- `w`：窗口长度，默认 `20`。

## 对齐说明

一二阶系数组合分类对齐 Python `bar_polyfit_V240428`。
