# jcc_two_crow_V221108：两只乌鸦

> 模块: `jcc.rs` | 类别: `kline`

## 参数模板

`"{freq}_D{di}K_两只乌鸦`

## 信号逻辑

1. 第一根长阳；
2. 第二根高开低走且收在第一根高点之上；
3. 第三根阴线继续下压，判 `看空`。

## 信号列表示例

- `Signal('60分钟_D1K_两只乌鸦_看空_任意_任意_0')`
- `Signal('60分钟_D1K_两只乌鸦_其他_任意_任意_0')`

## 参数说明

- `di`：倒数第 `di` 根K线，默认 `1`。

## 对齐说明

与 Python `jcc_two_crow_V221108` 一致。
