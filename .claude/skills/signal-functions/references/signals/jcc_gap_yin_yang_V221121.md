# jcc_gap_yin_yang_V221121：跳空并列阴阳

> 模块: `jcc.rs` | 类别: `kline`

## 参数模板

`"{freq}_D{di}K_并列阴阳`

## 信号逻辑

1. 最近三根满足跳空窗口；
2. 两根并列阴阳实体方差小于阈值；
3. 判定 `向上跳空/向下跳空`。

## 信号列表示例

- `Signal('15分钟_D1K_并列阴阳_向上跳空_任意_任意_0')`
- `Signal('15分钟_D1K_并列阴阳_向下跳空_任意_任意_0')`

## 参数说明

- `di`：倒数第 `di` 根K线，默认 `1`。

## 对齐说明

与 Python `jcc_gap_yin_yang_V221121` 一致。
